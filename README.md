## [](https://github.com/SamukaWenceslau/backend-test-java#pushpin-objetivo)📌  Objetivo

Estimular o conhecimento através da criação de uma API REST para gerenciar um estacionamento de carros e motos.

## Teste você mesmo...
Para testar o código é simples, você tem duas opções: 
1. **Testar localmente -** Você pode clonar este repositório, ou se preferir, você pode baixa-lo através [**deste link.**](https://github.com/SamukaWenceslau/backend-test-java/releases/tag/v1.0)

> ### Antes de testar remotamente, leia a documentação do projeto

2. **Testar remotamente** - Você pode testar os endpoints, utilizando o postman ou qualquer outro de sua preferência. Ou até mesmo na própria documentação interativa feita com swagger. 

### :point_right: [**Documentação**](https://project-parking.herokuapp.com/swagger-ui.html)

## :warning: Importante

Os endpoints são protegidos, sendo assim é necessário realizar o cadastro de uma nova empresa e efetuar o login. Você também pode utilizar o usuário ***default*** abaixo 👇, basta apenas fazer o login e pegar o token para acessar os demais endpoints da aplicação.

```JSON 
{  
	"email":  "parking@mail.com",  
	"password":  "123456"  
}
```
